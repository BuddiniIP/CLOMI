# 🌱 CLOMI - Close To Me

AI-powered habit tracking and wellness companion built with Android (Java).

## 📱 About

CLOMI helps users build healthy habits through personalized tracking, daily goals, AI-powered insights, and a virtual plant companion that grows as users stay consistent.

## ✨ Features

- 🌱 Personalized onboarding
- 💧 Water tracking
- 😴 Sleep tracking
- 🌸 Skincare tracking
- 😊 Mood tracking
- ⭐ XP & Level System
- 🔥 Daily streaks
- 🌿 Virtual plant growth
- 🤖 AI wellness assistant
- 📊 Reports & Analytics
- 🔔 Smart reminders

## 🛠️ Tech Stack

- Java
- Android Studio
- Material Design 3
- SharedPreferences
- Room Database (Coming Soon)
- Git & GitHub
